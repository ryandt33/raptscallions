# E03-T005: Classes service and API routes

## Description

Implement ClassesService and REST routes for class management and roster operations (add/remove students and teachers).

## Acceptance Criteria

- [ ] AC1: ClassesService with CRUD methods
- [ ] AC2: Methods: addMember, removeMember, getRoster, getStudents, getTeachers
- [ ] AC3: POST /classes - create class (teacher+)
